# YouTube PhotoTouch

이미지 밝기 조절 웹 애플리케이션

## 기능 / Features

- 이미지 업로드 / Image Upload
- 밝기 조절 (-150 ~ +150) / Brightness Adjustment
- 이미지 다운로드 / Image Download

## 사용 방법 / How to Use

1. 이미지를 업로드합니다 / Upload an image
2. 밝기 슬라이더로 조절합니다 / Adjust brightness with slider
3. 다운로드 버튼으로 저장합니다 / Download the edited image

## 기술 스택 / Tech Stack

- HTML5
- JavaScript (Vanilla)
- Canvas API
